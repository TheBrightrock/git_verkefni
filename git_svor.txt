Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	
	cd: Breytir hvar command line er að vinna.
	cd ~: Breytir working directory í "Home".
	ls: "Skrifar öll skjöl sem eru í working directoy.
	pwd: Skrifar working directory.
	mkdir: Býr til Folder.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?

	git clone: Aftritar skjal.
	git log: Sýnir hvað er búið að gera í git repository.
	git status: Sýnir hvernig aðstæður verkefnið er í.
	git diff: Sýnir mismun á tvemur branches.
	git checkout: Skiptir á milli git greinum.

3. Hver er munurinn á eftirfarandi git skipunum:
	
	a) git diff: Sýnir alla mismunni á tvemur greinum
	b) git diff --staged: Sýnir mismun á síðasta commit og núverandi stöðu. 
	c) git diff commit1 commit2: Sýnir mismun á ákveðum commit.

4. Hvað er version control, hverjir eru helstu kostir við að nota GIT?

Í git er version control sem leyfir notendanum að leita í gegnum eldri útgáfur af commit.
