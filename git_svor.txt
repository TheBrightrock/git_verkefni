Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	
	cd: Breytir hvar command line er að vinna.
	cd ~: Breytir working directory í "Home".
	ls: "Skrifar öll skjöl sem eru í working directoy.
	pwd: Skrifar working directory.
	mkdir: Býr til Folder.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?

	git clone: Aftritar skjal.
	git log: Sýnir hvað er búið að gera í git repository.
	git status: Sýnir hvernig aðstæður verkefnið er í.
	git diff: Sýnir mismun á tvemur branches.
	git checkout: Skiptir á milli git greinum.
